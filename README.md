# AI-Powered Automated Programming Test Evaluator

## Description
This project uses Artificial Intelligence to automatically evaluate programming tests based on correctness, logic, and performance.

## Features
- Automatic code evaluation
- AI-based scoring system
- Flask web interface
- Supports multiple test cases

## Technologies Used
- Python
- Flask
- Machine Learning
- HTML, CSS

## How to Run
1. Clone the repository
2. Install requirements:
   pip install -r requirements.txt
3. Run the application:
   python app.py
